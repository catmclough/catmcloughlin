The Book Atlas

https://book-atlas.com

When I go on vacation I like to take a book that's set in or near my destination. So, inspired by the NPR Book Concierge and Daunt Books, I created a personal database of books organized by setting or other associated locations.

The books are currently curated by myself, but this is a work in progress! I'd like to add support for cities, states and regions, a better experience for adding new books including book covers, and links to purchase any given book.

This app served as my introduction to React. It uses a Firebase DB and the Google Books API for book creation.

Frogger Soccer

Click here to play

I created this game in 2013 for a class at NYU called Interactive Computing. The code is written in ProcessingJS (now p5.js).

Frogger Soccer is a chase game where you are the ball and your goal is to find the net. If you run into a defender, you've been tackled and have to start at the beginning. See how many goals you can score before being tackled five times.

This was the first game I ever created. The challenge lied in learning how to track the movement of elements in a Processing program and handle collisions between them. This was also my first time creating a game loop and handling score tracking and resetting after a game has ended.

The defenders are original drawings by myself, done using 'make 8-bit art' by Jen Schiffer.

Spooky Halloween Game

Click here to play

Another collision-based game done for the mid-term of my Interactive Computing class at NYU. This one uses mouse-tracking and was completed in-class in about an hour.

Avoid the witch and the ghost, which appear at random y-coordinates and fly through the screen while you try to collect more candy corn than they do!

Space Consumers

View on Github

I made this weird little game in 2012 as a project for my CS101 class at NYU. The class taught OO concepts in Java and this project was meant to display my understanding of interfaces. It uses the Processing library's PApplet to draw the visualizations as well as the Minim library to make cool space sounds.

In the game, you are a black hole that consumes other space things as you move around. If you'd like to play you can follow the instructions in the README to run the project locally.
